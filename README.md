# Customer Sales Analysis — SQL

## Project Overview
This project analyses customer orders and sales data for a small retail business. The goal was to identify top-performing products, track monthly revenue trends and understand customer purchasing behaviour.

## Business Questions Answered
- Which products generate the most revenue?
- Which months have the highest and lowest sales?
- Who are the top 5 customers by total spend?
- What is the average order value per customer?
- Which product categories are underperforming?

## Tools Used
- SQL (SQLite)
- DB Browser for SQLite

## Files
- `schema.sql` — creates the database tables
- `data.sql` — inserts sample data
- `analysis.sql` — all analysis queries with comments

## Key Findings
- Hampers category generated the highest revenue across all months
- December consistently showed the highest order volume
- Top 5 customers accounted for approximately 40% of total revenue
- Average order value was R850 across all transactions
